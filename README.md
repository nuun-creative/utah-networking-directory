# Utah Business Networking Events Directory

A curated weekly directory of 80+ business networking events across Utah, sourced from 40+ organizations including chambers of commerce, Silicon Slopes, Meetup, Eventbrite, BNI, SCORE, and more.

## Features

- **80+ events** across Salt Lake County, Utah County, Weber County, Davis County, Cache County, Tooele County, Summit County, and Southern Utah
- **Filters** by region, event type, industry, cost, and recurring events
- **Full-text search** across titles, organizers, locations, and descriptions
- **Founder List** email signup for weekly digest with lifetime access offer
- **Dark theme** with responsive design

## Tech Stack

- React 18
- Vite
- Deployed on Cloudflare Pages

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Connected to Cloudflare Pages via GitHub. Pushes to `main` trigger automatic deployment.

- **Build command:** `npm run build`
- **Build output directory:** `dist`
- **Node version:** 18+
